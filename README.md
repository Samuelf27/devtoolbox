## 📌 Sobre o projeto

O **DevToolbox** reúne 9 utilitários essenciais do dia a dia de quem programa, num app rápido, offline-friendly e 100% no navegador (nenhum dado sai da sua máquina).

## 🧰 Ferramentas incluídas

| Ferramenta | O que faz |
|---|---|
| **JSON** | Formata, minifica e valida com mensagens de erro |
| **Base64** | Codifica/decodifica texto (com UTF-8) |
| **JWT** | Decodifica header e payload de um token |
| **Hash** | Gera SHA-1, SHA-256 e SHA-512 (Web Crypto API) |
| **UUID** | Gera UUIDs v4 em lote |
| **Timestamp** | Converte Unix ↔ data legível |
| **URL** | Encode/decode de componentes de URL |
| **Cor** | Converte HEX ↔ RGB ↔ HSL |
| **Lorem** | Gera texto fictício para protótipos |

## ✨ Destaques técnicos

- 🔒 **Privacidade total** — tudo roda no cliente; nada é enviado a servidores
- 🧩 Arquitetura por "registro de ferramentas" — fácil adicionar novas
- 🔗 Deep-link por hash (`#json`, `#hash`...) para compartilhar a aba
- 📋 Copiar com um clique e *toast* de feedback
- 📱 Responsivo com sidebar retrátil

## 🛠️ Tecnologias

HTML5 · CSS3 · JavaScript (Web Crypto API, Clipboard API) — sem frameworks
